# CupkekGames Transforms

Transform helpers extracted from HeroManager. Single static helper class today (`TransformUtils.SetScaleRecursive`); namespace `CupkekGames.Transforms` is pluralized to leave room for the API to grow without colliding with the `TransformUtils` class name.

## What's inside

**Runtime** (`CupkekGames.Transforms.asmdef`)
- `TransformUtils.SetScaleRecursive(Transform, Vector3)` — apply a scale to a transform and all descendants

## Dependencies

None.
